British Airways Data Analysis Project

Project Overview

This project involved analyzing customer feedback data for British Airways to identify trends, improve customer satisfaction, and enhance operational efficiency.

Key Objectives

  • Data Collection: Gather customer feedback from surveys and social media platforms.
  • Data Cleaning: Process and clean the data to ensure accuracy and consistency.
  • Analysis: Identify key trends and areas for improvement.
  • Visualization: Create visualizations to present findings effectively.

Results

The analysis revealed key pain points in customer service and provided actionable insights to improve the overall customer experience.
